iptables管理单个ip以及ip段
06-26-2014

屏蔽单个IP: iptables -I INPUT -s 123.45.6.7 -j DROP
屏蔽封123.0.0.1到123.255.255.254: iptables -I INPUT -s 123.0.0.0/8 -j DROP
屏蔽封123.45.0.1到123.45.255.254: iptables -I INPUT -s 123.45.0.0/16 -j DROP
屏蔽封123.45.6.1到123.45.6.254: iptables -I INPUT -s 123.45.6.0/24 -j DROP
解封: iptables -I INPUT -s 192.168.1.138 -j ACCEPT
